Je suis consultant Data chez Abylsen depuis juin 2024 à Metz.  
Je travaille en CDI et je suis en mission chez eMotors.  
J’analyse les besoins métiers dans plusieurs domaines : achats, finances, production, qualité et supply chain.  
Je collecte des données provenant d'automates industriels avec le protocole OPC UA.  
Je développe des ETL en Python, SQL et Spark, notamment sur Databricks.  
Je modélise les données avec une architecture en médaillon : Bronze, Silver et Gold.  
J’automatise les workflows ETL avec des planifications batch sur des machines virtuelles ou via Databricks Jobs.  
Je gère le versioning de mes projets sur Bitbucket.  
Je crée et maintiens des tableaux de bord Power BI pour aider à la décision.  
Je rédige aussi la documentation technique et des guides utilisateurs.  
Mes compétences incluent Python, SQL, Spark, Power BI, Azure, AWS S3, Bitbucket, VSCode, SharePoint, OPC UA, PostgreSQL et Datalake ERP.

J’ai aussi rédigé la documentation fonctionnelle et technique.  
Sur un autre projet, j’ai développé une application web pour l’analyse de données.  
J’ai construit une interface interactive dans Jupyter Notebook pour intégrer du code nCodeDS dans PySpark.  
J’ai développé un plugin PySpark pour automatiser l’extraction de données complexes.  
Mes outils de travail comprenaient GCP (BigQuery, Cloud Function, Pub/Sub, Cloud Storage, Scheduler, Git Repository), Python, SQL, Shell, HTML, CSS, JavaScript, Git, API REST, CI/CD, Looker Studio, Dataiku DSS et Jupyter Notebook.

J’ai fait un stage de Data Scientist chez Clemessy de mars 2022 à août 2022 à Mulhouse.  
J’ai travaillé sur un projet appelé SITG qui portait sur les tunnels de Lyon.  
J’ai recherché des règles d’association entre alarmes d’équipements.  
J’ai développé une application Dash/Plotly pour présenter le projet et suivre son avancement.  
J’ai traité les données d’alarme provenant d’une base PostgreSQL pour l’algorithme Apriori.  
J’ai mis en place une stratégie de filtrage basée sur l’analyse de graphes et la décomposition par composants.  
J’ai créé une interface interactive avec Voila pour des démonstrations en direct.  
Mes compétences durant ce stage incluaient Python, SQL, Dash, Plotly, Voila et PostgreSQL.  
J’ai aussi des connaissances en Linux, Docker et Jupyter Notebook.
